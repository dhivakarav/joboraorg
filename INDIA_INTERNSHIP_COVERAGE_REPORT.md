## How applications work (Track & Apply)

- Every job stores: `apply_url`, `source_name`, `source_domain`, `company`, `title`, `location`, `salary/stipend`, and internship/fresher/full-time classification.
- No auto-apply for these sources → Jobora exposes the **direct Apply URL**, opens the employer page in a **new tab**, and **tracks the application status** in the Activity Log.
- Ranking tiers: **T1** India intern/fresher (on-site) → **T2** remote-India intern/fresher → India (any) → **T3** global remote → global on-site.

## Coverage ceiling & next lever

Public ATS boards are full-time-weighted, so India internship volume is inherently modest here. The path to Naukri/Indeed-scale internship volume is the **keyed India aggregators** already wired in:
- **JSearch** (RapidAPI) and **Adzuna**/**Jooble** (`country=in`) — internship/fresher-aware, just add a free key.
- **Internshala** — needs an authorized feed (no public API); currently Mock Mode.
